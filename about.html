THIS IS THE ABOUT PAGE

This project is a School Project Hub where users can search for any project by name or student. All project data is stored in a separate JSON/JS file which the website reads and displays dynamically. The search button filters projects instantly on the page. Since this is a static web project, new projects cannot be saved directly from the browser — any updates to projects need to be added manually in the JSON/JS file. However, if this project is uploaded to a web server, it will display all the updated projects immediately. The background animation and responsive layout make the interface more engaging for viewers.

This is a simple website I built for our school exhibition where all the projects created by students can be stored, searched, and displayed in one place. Let me explain how it works.

    1. The Website Page
    • I created the main page using HTML and CSS.
    • - HTML is used to structure the content.
    • - CSS is used to design and style the page, making it colorful, modern, and easy to use.
    • - I also added a particle animation background using JavaScript to make the page look attractive.
  1. 2. The Data Storage
    • All the project details, like the project name, summary, student’s name, class, and cost, are stored in a separate JSON file. JSON stands for JavaScript Object Notation. It is like a digital notebook that stores information in a very organized way.
  2. 3. How Search Works
    • When a user types a project name or student’s name in the search box and clicks the button, the program checks the JSON file, finds the matching project, and displays it in the form of a card. Each card shows the project’s details clearly so that anyone can understand it quickly.
  3. 4. Why this is Useful
    • - Instead of flipping through papers or registers, teachers and students can simply search and find any project instantly.
    • - It is reusable — every year we can just add new projects into the JSON file.
    • - It also teaches us the importance of digital record-keeping and technology in education.
